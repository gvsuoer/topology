Task 8.d.

Let \(E^m = \{(x_n) \in H \mid x_k = 0 \text{ for } k > m\}\text{.}\) Let \(f: E^m \to \R^m\) be defined by \(f((x_n)_{n=1}^{\infty}) = (x_n)_{n=1}^m\text{.}\) Show that \(f\) is a bijection such that \(d((x_n), (y_n)) = d_E(f((x_n)), f((y_n)))\) for any elements \((x_n)\text{,}\) \((y_n)\) in \(H\text{.}\) So \(E^m\) is essentially the same as \(\R^m\) and so we can consider the space \(\R^m\) as embedded in \(H\) as a subspace of \(H\) for every \(m \in \Z^+\text{.}\)
