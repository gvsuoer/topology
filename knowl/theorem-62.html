Theorem 19.15.

Let \(X\) be a finite topological space. Then \(X\) is connected if and only if \(X\) is path connected.

Proof.

Let \(X\) be a finite topological space. TheoremÂ 19.10 demonstrates that if \(X\) is path connected, then \(X\) is connected. For the reverse implication, assume that \(X\) is path connected. Then \(X\) is composed of a single path component, \(P=X\text{.}\) Since the path components and components of \(X\) are the same, we conclude that \(P= X\) is a component of \(X\) and that \(X\) is connected.
