Lemma 12.3.
Let \(X\) be a set and \(\B\) a collection of subsets of \(X\) such that
  1. For each \(x \in X\text{,}\) there is a set in \(\B\) that contains \(x\text{.}\)
  2. If \(x \in X\) is an element of \(B_1 \cap B_2\) for some \(B_1, B_2 \in \B\text{,}\) then there is a set \(B_3 \in \B\) such that \(x \in B_3 \subseteq B_1 \cap B_2\text{.}\)
Let \(B_1\text{,}\) \(B_2\text{,}\) \(\ldots\text{,}\) \(B_n\) be in \(\B\text{.}\) If \(x \in \bigcap_{1 \leq k \leq n} B_k\text{,}\) then there is a set \(B \in \B\) such that \(x \in B\) and \(B \subseteq \bigcap_{1 \leq k \leq n} B_k\text{.}\)
