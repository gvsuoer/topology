Item 1.

For each of the given sets \(A\text{,}\) find all boundary points, limit points, and isolated points. Then determine if the set \(A\) is a closed set in the metric space \((X,d)\text{.}\) Explain your reasoning.
  • \(X = \R\text{,}\) \(d = d_E\text{,}\) the Euclidean metric, \(A = [0,0.5)\text{.}\)
  • \(X = \{x \in \R \mid 0 \lt x \leq 1\}\text{,}\) \(d = d_E\text{,}\) the Euclidean metric, \(A = (0,0.5]\text{.}\)
  • \(X = \{a,b,c,e\}\text{,}\) \(d\) is the discrete metric defined by
    \begin{equation*} d(x,y) = \begin{cases}0 \amp \text{ if } x = y \\ 1 \amp \text{ if } x \neq y, \end{cases} \end{equation*}
    and \(A = \{a,b\}\text{.}\)
