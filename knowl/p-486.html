We have only shown that \(d_T\) and \(d_M\) are metrics on \(\R^2\text{,}\) but similar arguments apply in \(\R^n\text{.}\) Proofs are left to Exercise 5 and Exercise 6. In addition, the discrete metric
\begin{equation*} d(x,y) = \begin{cases}0 \amp \text{ if } x=y \\ 1 \amp \text{ if } x \neq y \end{cases} \end{equation*}
makes any set \(X\) into a metric space. The proof is left to Exercise 1.
