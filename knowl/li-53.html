Item.

If \(X\) and \(Y\) are sets, then the union \(X \cup Y\) is the set
\begin{equation*} X \cup Y = \{z \mid z \in X \text{ or } z \in Y\}\text{.} \end{equation*}
The union of an arbitrary collection \(\{X_{\alpha}\}\) of sets for \(\alpha\) in some indexing set \(I\) is the set
\begin{equation*} \bigcup_{\alpha \in I} X_{\alpha} = \{z \mid z \in X_{\beta} \text{ for some } \beta \in I\}\text{.} \end{equation*}
