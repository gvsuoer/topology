If \(X\) and \(Y\) are sets, then the intersection \(X \cap Y\) is the set
\begin{equation*} X \cap Y = \{z \mid z \in X \text{ and } z \in Y\}\text{.} \end{equation*}
The intersection of an arbitrary collection \(\{X_{\alpha}\}\) of sets for \(\alpha\) in some indexing set \(I\) is the set
\begin{equation*} \bigcap_{\alpha \in I} X_{\alpha} = \{z \mid z \in X_{\beta} \text{ for all } \beta \in I\}\text{.} \end{equation*}
