Let \(X\) be the set of continuous functions \(f: [a,b] \to \mathbb{R}\text{.}\) Let \(d^*\) be the distance function on \(X\) defined by
\begin{equation*} d^*(f,g)= \int_a^b \la f(t)-g(t) \ra \, dt\text{,} \end{equation*}
for \(f, g \in X\text{.}\) For each \(f \in X\text{,}\) set
\begin{equation*} I(f) = \int_a^b f(t) \, dt\text{.} \end{equation*}
