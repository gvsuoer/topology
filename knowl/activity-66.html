  • Let \(X=\{a,b,c,d,e,f\}\) and \(\tau= \{\emptyset, \{a\}, \{c,d\},\{a,c,d\}, \{b,c,d,e,f\}, X \}\text{.}\)
    1. Is the set
      \begin{equation*} \mathcal{B} = \{\{a\},\{a,c,d\} \} \end{equation*}
      a basis for \(\tau\text{?}\) If not, add the smallest number of sets that you can to \(\mathcal{B}\) to make a basis for this topology.
    2. Is the set
      \begin{equation*} \mathcal{B} = \{\{a\},\{c,d\},\{b,c,d,e,f\} \} \end{equation*}
      a basis for \(\tau\text{?}\) If not, add the smallest number of sets that you can to \(\mathcal{B}\) to make a basis for this topology.
  • Let \(X=\{a,b,c\}\) and let \(X\) have the discrete topology (the topology consisting of all subsets of \(X\)). Is \(\mathcal{B} = \{\{a\},\{c\},\{a,b\},\{b,c\} \}\) a basis for \(\tau\) in the the discrete topology? If not, add the smallest number of sets that you can to \(\mathcal{B}\) to make a basis for this topology.
  • Find a basis for the discrete topology on any set \(X\text{.}\)
