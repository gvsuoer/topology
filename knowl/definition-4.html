Definition 2.2.

Let \(f\) be a function from a set \(A\) to a set \(B\text{.}\)
  1. The function \(f\) is an injection if, whenever \((a,b)\) and \((a',b)\) are in \(f\text{,}\) then \(a=a'\text{.}\) Alternatively, using the function notation, \(f\) is an injection if \(f(a)=f(a')\) implies \(a=a'\text{.}\)
  2. The function \(f\) is a surjection if, whenever \(b \in B\text{,}\) then there is an \(a \in A\) so that \((a,b)\) is in \(f\text{.}\) Alternatively, using the function notation, \(f\) is a surjection if for each \(b \in B\) there exists an \(a \in A\) so that \(f(a)=b\text{.}\)
  3. The function \(f\) is a bijection if \(f\) is both an injection and a surjection.
