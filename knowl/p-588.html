If a subset \(S\) of \(\R\) has a lower bound, we say that \(S\) is bounded below. So the set \(T = \{d_E(1,a) \mid a \in (-1,0)\}\) is bounded below by 1. The set \(T\) is also bounded below by 0.5 and 0.
