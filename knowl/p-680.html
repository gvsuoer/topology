Let’s deal with the first statement, making the values of \(f(x)\) “as close to \(f(a)\) as we want”. What this means is that if we set any tolerance, say \(0.0001\text{,}\) then we can make the values of \(f(x)\) within \(0.0001\) of \(f(a)\text{.}\) Since the absolute value \(| f(x)-f(a) |\) measures how close \(f(x)\) is to \(f(a)\text{,}\) we can rewrite the statement that the values of \(f(x)\) are within \(0.0001\) of \(f(a)\) as \(| f(x) - f(a) | \lt 0.0001\text{.}\) Of course, \(0.0001\) may not be as close as we want to \(f(a)\text{,}\) so we need a way to indicate that we can make the values of \(f(x)\) arbitrarily close to \(f(a)\) — within any tolerance at all. We do this by making the tolerance a parameter, \(\epsilon\text{.}\) Then our job is to make the values of \(f(x)\) within \(\epsilon\) of \(f(a)\) regardless of the size of \(\epsilon\text{.}\) We write this as
\begin{equation*} | f(x) - f(a) | \lt \epsilon\text{.} \end{equation*}
