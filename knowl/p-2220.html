As an example to motivate the definition, consider the sets \(X = (0,1) \cup (1,2)\) and \(Y = [1,2]\) in \(\R\) with the Euclidean topology. Notice that we can write \(X\) as the union of two disjoint open sets \(X_1 = (0,1)\) and \(X_2 = (1,2)\text{.}\) So we shouldnâ€™t think of \(X\) as being connected. However, if we attempt to write \(Y\) as a union of two subsets, say \(Y_1 = [1,1.5)\) and \(Y_2 = [1.5,2]\text{,}\) it is impossible for both of these subsets to be open and disjoint. So \(Y\) is a set we should consider to be connected. This is the notion of connectedness that we wish to investigate.
