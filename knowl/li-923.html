Item 2.

Let \(a\) and \(b\) be integers with \(b \neq 0\text{.}\) Let \(A_{a,b} = a\Z+b = \{a + kb \mid k \in \Z\}\text{.}\)
  • Show that \(\{A_{a,b} \mid a, b \in \Z, b \neq 0\}\) is a basis for a topology \(\tau\) on \(\Z\text{.}\) Hint.
  • Let \(f : \Z \to \Z\) be defined by \(f(n) = n + (-1)^n\text{.}\)
    1. Prove that \(f\) is a bijection.
    2. If \(O\) is an open set in \(\Z\text{,}\) is \(f(O)\) an open set?
    3. If \(U\) is an open set in \(\Z\text{,}\) is \(f^{-1}(U)\) an open set? Hint.
