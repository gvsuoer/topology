Exercise 7.

If \(x = (x_1, x_2, \ldots, x_n)\text{,}\) we let \(|x| = \sqrt{x_1^2+x_2^2+ \cdots + x_n^2}\text{.}\) For \(x = (x_1, x_2, \ldots, x_n)\) and \(y = (y_1, y_2, \ldots y_n)\text{,}\) define \(d_H: \R^n \times \R^n \to \R\) by
\begin{equation*} d_H(x,y) = \begin{cases}0 \amp \text{ if } x=y \\ |x|+|y| \amp \text{ otherwise } . \end{cases} \end{equation*}

(a)

Show that \(d_H\) is a metric (called the hub metric).

(b)

(i)
Let \(a = \left(\frac{1}{2}, 0\right)\text{.}\) Explicitly describe which points are in the set \(B(a,1)\) in \((\R^2, d_H)\text{.}\) (See ExerciseÂ 2 for the definition of an open ball.)
(ii)
Let \(a = (3,4)\text{.}\) Explicitly describe which points are in the set \(B(a,1)\) in \((\R^2, d_H)\text{.}\)
(iii)
Now explicitly describe all open balls in \((\R^2, d_H)\text{.}\)
