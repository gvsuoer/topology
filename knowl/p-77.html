We can visualize these sets using Venn diagrams. A Venn diagram is a depiction of sets using geometric figures. For example, if \(U\) is a set containing all other sets of interest (we call \(U\) the universal set), we can represent \(U\) as a large container (say a rectangle) with subsets \(A\) and \(B\) as smaller containers (say circles), and shade the elements in a given set.
