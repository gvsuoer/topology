Item.

A topology on a set \(X\) is a collection of open subsets of \(X\text{.}\) More specifically, a set \(\tau\) of subsets of a set \(X\) is a topology on \(X\) if
  1. \(X\) and \(\emptyset\) belong to \(\tau\text{,}\)
  2. any union of sets in \(\tau\) is a set in \(\tau\text{,}\) and
  3. any finite intersection of sets in \(\tau\) is a set in \(\tau\text{.}\)
