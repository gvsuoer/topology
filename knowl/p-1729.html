We might consider two metric spaces \((X, d_X)\) and \((Y, d_Y)\) to be equivalent if we can find a bijection between the two sets \(X\) and \(Y\) that preserves the metric properties. That is, find a bijective function \(f : X \to Y\) such that \(d_X(a,b) = d_Y(f(a), f(b))\) for all \(a,b \in X\text{.}\) In other words, \(f\) preserves distances.
