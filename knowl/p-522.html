If \(x = (x_1, x_2, \ldots, x_n)\text{,}\) we let \(|x| = \sqrt{x_1^2+x_2^2+ \cdots + x_n^2}\text{.}\) For \(x = (x_1, x_2, \ldots, x_n)\) and \(y = (y_1, y_2, \ldots y_n)\text{,}\) define \(d_H: \R^n \times \R^n \to \R\) by
\begin{equation*} d_H(x,y) = \begin{cases}0 \amp \text{ if } x=y \\ |x|+|y| \amp \text{ otherwise } . \end{cases} \end{equation*}
