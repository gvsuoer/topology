Task 18.11.b.

Let \(D = \{(x,y) \mid x^2+y^2 \leq 4\}\) in \((\R^2, d_E)\text{.}\) That is, \(D\) is the closed disk of radius \(2\) in the plane. The set \(D\) has no cut points. However, if \(S = \{(x,y) \mid x^2+y^2 = 1\}\text{,}\) then \(D \setminus S\) consists of two connected components: the open ball \(B((0,0),1)\) and the annulus \(\{(x,y) \mid 1 \lt x^2+y^2 \leq 4\}\) as illustrated in FigureÂ 18.12. So \(S\) is a cut set of \(D\text{.}\)
