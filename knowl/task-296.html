Task 6.a.

We will first want to express \(f(x)g(x) - f(a)g(a)\) in a more useful way. Use the fact that \(f(x) = f(a) + (f(x)-f(a))\) and \(g(x) = g(a) + (g(x)-g(a))\) to show that
\begin{align*} f(x)g(x)-f(a)g(a) = f(a)(g(x)-g(a)) \amp + g(a)(f(x)-f(a))\\ \amp + (f(x)-f(a))(g(x)-g(a))\text{.} \end{align*}
