Since the open interval \((a-\delta, a+\delta)\) can be described completely by the Euclidean metric as the set of \(x\) values so that \(d_E(x,a) \lt \delta\text{,}\) there is no reason why we can’t extend this notation of open interval to any metric space. We must note, though, that \(\R\) is one-dimensional while most metric spaces are not, so the term “interval” will no longer be appropriate. We replace the concept of interval with that of an open ball.
