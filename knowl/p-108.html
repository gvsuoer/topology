Let \(\{A_{\alpha}\}\) is a collection of sets indexed by a set \(I\) in some universal set \(U\text{.}\) Then
  1. \(\displaystyle \displaystyle \left(\bigcup_{\alpha \in I} A_{\alpha}\right)^c = \bigcap_{\alpha \in I} A_{\alpha}^c\)
  2. \(\displaystyle \displaystyle \left(\bigcap_{\alpha \in I} A_{\alpha}\right)^c = \bigcup_{\alpha \in I} A_{\alpha}^c\)
