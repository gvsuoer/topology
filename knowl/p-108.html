Let \(A\text{,}\) \(B\text{,}\) and \(C\) be subsets of a universal set \(U\text{.}\)
Properties of the Empty Set
  1. \(\displaystyle A \cap \emptyset = \emptyset\)
  2. \(\displaystyle A \cup \emptyset = A\)
  3. \(\displaystyle A-\emptyset = A\)
  4. \(\displaystyle \emptyset^c = U\)
Properties of the Universal Set
  1. \(\displaystyle A \cap U = A\)
  2. \(\displaystyle A \cup U = U\)
  3. \(\displaystyle A-U = \emptyset\)
  4. \(\displaystyle U^c = \emptyset\)
Idempotent Laws
  1. \(\displaystyle A \cap A = A\)
  2. \(\displaystyle A \cup A = A\)
Commutative Laws
  1. \(\displaystyle A \cap B = B \cap A\)
  2. \(\displaystyle A \cup B = B \cup A\)
Associative Laws
  1. \(\displaystyle (A \cap B) \cap C = A \cap (B \cap C)\)
  2. \(\displaystyle (A \cup B) \cup C = A \cup (B \cup C)\)
Distributive Laws
  1. \(\displaystyle A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\)
  2. \(\displaystyle A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\)
Basic Properties
  1. \(\displaystyle \left(A^c\right)^c = A\)
  2. \(\displaystyle A - B = A \cap B^c\)
Subsets and Complements
\(\displaystyle A \subseteq B \text{ if and only if } B^c \subseteq A^c\)
