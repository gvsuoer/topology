Definition 17.2.

A cover of a subset \(A\) of a topological space \(X\) is a collection \(\{S_{\alpha}\}\) of subsets of \(X\) for \(\alpha\) in some indexing set \(I\) so that \(A \subseteq \bigcup_{\alpha \in I} S_{\alpha}\text{.}\) In addition, if each set \(S_{\alpha}\) is an open set, then the collection \(\{S_{\alpha}\}\) is an open cover for \(A\text{.}\)
